## 开始使用

1. 安装 magua-ui
    ```nodejs
        npm i -S magua-ui
    ```
2. 引入 magua-ui
    ```javascript
        import {GuaButton} from 'magua-ui'
        import 'magua-ui/dist/magua.css'
    ```
